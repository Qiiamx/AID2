# 资源文件说明

## 背景图片

请将背景图片放在 `bg/` 目录下，命名格式为：
- `bg1.png`
- `bg2.png`
- `bg3.png`
- ... 以此类推

图片要求：
- 建议分辨率：1920x1080 或更高
- 格式：PNG、JPG 均可
- 图片会在页面中自动轮播，每15秒切换一张

## 图标图片

请将身份图标放在 `icons/` 目录下，命名格式为：
- `icon-host.png` - 主持人图标
- `icon-team-a.png` - 队伍A图标
- `icon-team-b.png` - 队伍B图标
- `icon-viewer.png` - 观众图标

图片要求：
- 建议尺寸：64x64 像素
- 格式：PNG（支持透明背景）
- 如果图片不存在，会显示emoji占位符（👑、🅰️、🅱️、👁️）

## 注意事项

- 如果图片文件不存在，页面会使用占位符显示，不会报错
- 建议使用透明背景的PNG图标，效果更好
- 背景图片建议使用深色主题，与页面风格一致
